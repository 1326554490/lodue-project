# Lodue Project Agent Guide

## 项目定位

Lodue 是一个面向轻微阅读困难、阅读疲劳、注意力分散用户的陪伴式阅读系统原型。

它不是普通阅读器，而是一个包含：
- 阅读前校准
- 阅读中适应
- 阅读行为可视化陪伴
- 阅读后复盘
- 低打扰轻社交共读

的完整交互系统。

## 技术栈

- Vite
- React
- JavaScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Recharts
